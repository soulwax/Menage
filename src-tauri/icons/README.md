# Menage app icons

A 3×3 sprite-atlas grid on the family's dark violet: one tile gold (the one
being worked on), one tile a dashed outline (the one the cutter just took).

Generated procedurally (pure-Node PNG/ICO encoder — see the repo history for
`make_icons.mjs`). To regenerate from a new 512px master, either rerun a
script like it or use `npm run tauri icon <source.png>`.

The set is tracked in git — it is the app's identity.
